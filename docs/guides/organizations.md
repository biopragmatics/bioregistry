The responsible organization(s) can be annotated to a Bioregistry record to
enable higher-level summarization complementary to the contact person. This can
be useful to contextualize resources that are adverse to having single contact
person, e.g., from the NIH.

Some external registries that the Bioregistry imports already annotate
organizations, such as MIRIAM. Here are some tips for curating organizations
manually:

1. Search ROR using the domain from the homepage or URI format string in a
   Bioregistry record, since ROR records explicitly include domains
2. If a Bioregistry record has a primary contact, check the affiliation of the
   contact by looking at their ROR page
3. If a Bioregistry record has publications, check the affiliation of the
   primary/corresponding authors

Organizations can be annotated with one of the following identifiers:

1. Research Organization Registry (ROR)
2. Wikidata
3. Gemeinsame Normdatei (GND, Integrated Authority File)
